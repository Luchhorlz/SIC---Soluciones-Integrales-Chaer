## Lenguaje visual

- Interfaz clara, amplia y de baja densidad aparente.
- Bordó para navegación activa y acciones primarias; dorado para acentos.
- Tarjetas blancas, bordes grises suaves, radios moderados y sombras discretas.
- Iconografía lineal consistente.
- Cabeceras amplias, barras de búsqueda prominentes y sidebars para paneles.
- Estados siempre combinan color, texto e icono.
- Adaptación mobile-first: en resultados móviles, lista y mapa se alternan con pestañas.

## Ajustes obligatorios por consistencia del dominio

Las referencias son visuales, no autorizan requisitos incompatibles con el plan:

- La pantalla de contratación no debe cobrar el trabajo dentro de SIC en el MVP; se reemplaza el paso de pago por confirmación/coordinación.
- “Pagos protegidos/seguros” no se prometerá para trabajos fuera de plataforma.
- Los tres planes y precios ilustrados no se implementan hasta confirmación comercial; el valor inicial será configurable.
- Fotografías, mapas, certificados y datos personales mostrados son demostrativos y no se reutilizan como datos reales.
- La interfaz final usará español rioplatense neutral y accesibilidad WCAG 2.2 AA.
